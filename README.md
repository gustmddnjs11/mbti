# 관점 토크 (MBTI 관점 토론 앱)

> 같은 상황, 다른 생각 — MBTI 유형별 관점 차이로 함께 토론하는 웹앱.

## 실행
`index.html`을 브라우저에서 열면 바로 실행됩니다. (설치·빌드 불필요)

## 모드
- 🧍 **혼자** — 카드에 답하며 내 성향 알아보기
- 🧑‍🤝‍🧑 **함께** — 폰 하나로 번갈아 답하고 비교
- 💑 **MBTI 토론** — 두 유형 넣으면 둘이 갈릴 주제 찾아주기 (관계·엉뚱·공포·만약에·딜레마)
- 🎯 **서로 맞히기** — 상대 답을 맞히는 예측 게임 (+ 링크 공유 도전장)
- 📸 **인스타 카드** — 토론 주제를 1080×1080 인스타 이미지로

## 배포
GitHub Pages로 호스팅됩니다.
